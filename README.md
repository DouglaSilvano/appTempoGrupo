# AppTempo

Projeto feito em grupo durante o grupo de estudos da Preto no Branco pra puxar a previsão do tempo e jogar na tela através de uma API. A gente bate na API do OpenWeather e entrega o clima de agora, a previsão da semana e o detalhe por horário.

## O que tem no aplicativo?

* **Telão de Hoje (Esquerda):** Puxa a temperatura na hora, joga o ícone na tela e diz como está o tempo.
* **Smart search:** A partir de 3 letras da cidade, a barra já lista as opções. Assim que clicar em uma opção de cidade, ele atualiza o layout inteiro com a cidade selecionada.
* **Visão do Futuro (Cards da Direita):** O código busca no JSON da API e monta uns cards mostrando só o que interessa formando uma previsão de como estará o tempo no horário da tarde dos próximos dias.
* **Horário a Horário (Bloquinhos azuis):** Mostra os próximos 5 blocos de previsão.

## Tecnologias usadas:

* **A Stack:** HTML, CSS flexbox e JavaScript puro.
*  **API:** OpenWeatherMap.

# Como rodar o app:

1. Baixar o código em ZIP ou dar um `git clone repositório`
2. Abre o `index.html` no navegador.
3. Utilizar o programa
